# Excel 文件处理系统

一个在线 Excel 文件处理工具，可以：
1. 将总表拆分成各家医院的分表
2. 将总表拆分成职工和居民的汇总表

## 功能特点

- 完全在浏览器中处理，无需服务器
- 支持拖拽上传
- 自动处理字体和格式
- 批量导出为 ZIP 文件

## 使用方法

1. 访问 [在线工具地址](https://lylllii.github.io/biaogechaifen/)
2. 选择需要的功能（拆分各家分表/拆分汇总表）
3. 上传 Excel 文件
4. 点击处理按钮
5. 下载生成的 ZIP 文件

## 注意事项

- 上传的 Excel 文件必须包含特定的列名
- 建议使用最新版本的 Chrome、Firefox 或 Edge 浏览器
